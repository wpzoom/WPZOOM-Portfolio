## Features

### Free Features
- **Portfolio Posts** - Create dedicated portfolio items
- **Reusable Portfolio Layouts** - Create once, use anywhere
- **Portfolio Block** - Native Gutenberg block
- **Portfolio Layout Block & Shortcode** - Insert in any page builder (Elementor, Beaver Builder, etc.)
- **Blog Posts Support** - Works with regular blog posts too
- **Isotope Filtering Effect** - Category filter with smooth animations
- **Image Lightbox** - Beautiful popup gallery
- **3 Layouts**: Grid with Overlay, Masonry & Columns
- **Multiple Aspect Ratios** - Portrait, cinema, square, no crop
- **Customizable Colors & Typography**
- **100% GDPR Compliant** - No Google Fonts, no external requests

### PRO Features
- Video Integration
- Video Lightbox (YouTube, Vimeo & MP4)
- Video Background on Hover
- Portfolio Re-order
- Customize Colors
- Premium Support

## Installation

### From WordPress.org

1. Go to **Plugins > Add New** in your WordPress admin
2. Search for "WPZOOM Portfolio"
3. Click **Install Now** and then **Activate**

### Manual Installation

1. Download the plugin from [WordPress.org](https://wordpress.org/plugins/wpzoom-portfolio/)
2. Upload to `/wp-content/plugins/wpzoom-portfolio/`
3. Activate through the **Plugins** menu

### From GitHub

```bash
cd wp-content/plugins
git clone https://github.com/wpzoom/WPZOOM-Portfolio.git wpzoom-portfolio
```

## Usage

### Creating Portfolio Items

1. Go to **Portfolio** in your WordPress admin
2. Click **Add New** to create a portfolio item
3. Add title, content, featured image, and assign categories

### Displaying Your Portfolio

#### Using the Portfolio Block

1. Edit any page or post
2. Add the **Portfolio** block
3. Configure layout, columns, and filtering options
4. Customize colors and typography in the Style tab

#### Using Portfolio Layouts (Shortcode)

1. Go to **Portfolio > Portfolio Layouts**
2. Create a new layout and configure options
3. Copy the shortcode from block settings
4. Paste anywhere on your site

### Troubleshooting

#### 404 Error on Portfolio Pages

If you see a 404 error on single portfolio pages:
1. Go to **Settings > Permalinks**
2. Click **Save Changes** (no changes needed)
3. This flushes WordPress permalinks and fixes the issue
